# Modal Popup Window

A modern and responsive Modal Popup Window built using HTML, CSS, and JavaScript. This project demonstrates how to create an interactive popup with smooth animations, an overlay background, and user-friendly controls.

## 🚀 Features

* Open modal with a button click
* Overlay background effect
* Close modal using the close button
* Close modal by clicking outside the popup
* Smooth CSS animations and transitions
* Responsive design for all screen sizes
* Clean and easy-to-understand code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How It Works

1. Click the **Open Modal** button.
2. The popup window appears with a smooth animation.
3. View the message or interact with the form inside the modal.
4. Close the modal using:

   * The close (×) button
   * Clicking outside the modal area
